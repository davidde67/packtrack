## 🔧 Tools Needed (Pull from Storage)

### Power Tools (from storage locker in Alameda — CSI Storage, Hornet Dr)
- [ ] Table saw — PRIMARY tool, MUST have for cutting 4×4 posts to length
- [ ] Drill (cordless, charged) — predrill pilot holes in Douglas fir
- [ ] Impact driver (or use drill) — drive structural screws
- [ ] Circular saw (backup) — crosscuts
- [ ] Speed square — mark cut lines
- [ ] Tape measure (25 ft) — layout + marking
- [ ] Chalk line — mark bed positions on fabric
- [ ] Laser level or level (2 ft) — check bed is level

### Hand Tools
- [ ] Socket/wrench set — tighten bolts on L-brackets
- [ ] Level (2 ft) — check bed is level
- [ ] Shovel — positioning and backfill
- [ ] Rake — smooth fabric + gravel spreading
- [ ] Work gloves — handling lumber + gravel
- [ ] Safety glasses — table saw + drilling
- [ ] Ear protection — table saw

### Assembly Checklist (Order of Operations)
1. Cut all 4×4 posts to 24" length (16 total)
2. Pre-drill screw holes in all lumber
3. Assemble 4 bed frames on flat surface:
   - Bottom row of 2×12 boards, L-brackets at corners
   - Top row of 2×12 boards, L-brackets at corners
   - 4×4 corner posts sandwiched between
4. Carry beds to position on fabric
5. Check level, adjust as needed
